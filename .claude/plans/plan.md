## 5. 사용할 기술/API

### TCGdex API (기존 유지)
- GET /v2/ko/cards/{cardId} → 상세 정보
  - 신규 사용 필드: `illustrator`, `set.name`, `set.id`

### 팩 출현율 데이터 (신규)
- URL: https://raw.githubusercontent.com/flibustier/pokemon-tcg-pocket-database/main/dist/pullRates.json
- 캐시 키: `tcgp_pull_rates`
- TTL: 24시간 (기존 cacheGet/cacheSet 재사용)
- 데이터 구조 예시:
  ```
  {
    "A1": {
      "Regular Pack": {
        "appearance_rate": 99.95,
        "cards": 5,
        "slots": {
          "1": { "C": 100 },
          "2": { "C": 100 },
          "3": { "C": 100 },
          "4": { "RR": 5, "R": 26.6, "U": 21.4, "C": 47 },
          "5": { "UR": 0.04, "SR": 5.96, "RR": 13.5, "R": 38.5, "U": 21, "C": 21 }
        }
      },
      "Rare Pack": { ... }
    }
  }
  ```

### 레어도 코드 매핑 (TCGdex 심볼 → pullRates 코드)
| TCGdex 레어도 심볼 | pullRates 코드 | 설명 |
|--------------------|----------------|------|
| ◇ | C | 커먼 |
| ◇◇ | U | 언커먼 |
| ◇◇◇ | R | 레어 |
| ◇◇◇◇ | RR | 더블 레어 |
| ☆ | SR | 스타 레어 |
| ☆☆ | SSR | 더블 스타 |
| ☆☆☆ | UR | 트리플 스타 |
| ♛ | UR | 크라운 레어 |
| ✦ | AR | 일러스트 레어 |
| ✦✦ | IM | 스페셜 일러스트 레어 |

*주의: 실제 pullRates.json 데이터와 대조 후 코드 구현 시 조정 필요*

## 7. 예상 위험 요소
- 위험 요소 1: TCGdex `illustrator` 필드 없는 카드
  - 대응: 없을 시 일러스트레이터 섹션 숨김
- 위험 요소 2: TCGdex `set.name`/`set.id` 반환 불안정
  - 대응: 없을 시 카드 ID에서 setId 추출 (예: "A1-001" → "A1")
- 위험 요소 3: pullRates.json의 setId가 TCGdex setId와 불일치
  - 대응: 실제 데이터 대조 후 필요 시 매핑 테이블 추가. 없을 시 출현율 섹션 숨김
- 위험 요소 4: 레어도 코드 매핑 불완전 (신규 레어도 추가 시)
  - 대응: 매핑 없는 코드는 슬롯 표시 생략, 향후 보완
- 위험 요소 5: pullRates.json GitHub URL 변경/삭제
  - 대응: 데이터 없을 시 출현율 섹션 전체 숨김 (silent fail)
